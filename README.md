# xrpl-dev

## XRPL NFT Marketplace for the XRP Ledger
##      POC on XRP Ledger Devnet


## Build

Install nodejs <br />
npm install  <br />
npm install xrpl  <br />
npm install express  <br />

## Run
node xrpl-nft-seller.js  <br />

node xrpl-nft-buyer.js  <br />
